# AI Image Generation Prompts

Every image slot on the homepage loads a file from this folder. The PNGs currently
here are **illustrated placeholders**. To upgrade any slot to a real AI-generated
photo, generate an image with the prompt below (DALL·E, Midjourney, Firefly, etc.),
then save it over the same filename — the site picks it up automatically, no code
changes needed. (PNG format; keep roughly the same aspect ratio as the current file.)

**Style prefix — prepend to every prompt:**

> Premium product-lifestyle photography, bright natural light, clean modern
> aesthetic like an Apple ad, shallow depth of field, a small matte-graphite
> smart camera with a dark lens and a tiny green LED visible in the scene,
> photorealistic, no text or watermarks.
